# Aula-POO-01

Repositório criado para a disciplina de **Programação Orientada a Objetos**.

## Descrição

Este projeto contém exemplos e exercícios práticos de conceitos fundamentais de Programação Orientada a Objetos (POO) em Java.

## Tecnologia

- **Linguagem:** Java

## Objetivo

- Descrever uma cadeira em lógica de programação orientada a objeto.
- Com atributos para altura, quantidade de pés e material.
- Com métodos para elevar a altura e mudar o material.

## Como Usar

1. Clone este repositório
2. Abra o projeto na sua IDE preferida (IntelliJ, Eclipse, VS Code, etc.)
3. Compile e execute os arquivos `.java`
